#  Cinema Audience Forecasting Challenge  
**MLP Project – T3 2025**

##  Overview  
This project is part of the Kaggle competition **“Cinema Audience Forecasting Challenge”**, where the goal is to predict daily cinema audience counts for multiple theaters using historical booking data.

The dataset combines information from two platforms:  
- **BookNow** (online booking platform)  
- **CinePOS** (on-site POS ticketing system)

The task is to forecast how many people will attend a given theater on a given day.

Competition link:  
https://www.kaggle.com/competitions/Cinema_Audience_Forecasting_challenge
